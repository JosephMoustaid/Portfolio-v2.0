ClubXtreme - Sports Club Management Platform

Developed ClubXtreme, a comprehensive web platform for managing sports clubs. It features member management, event scheduling, transaction tracking, and training sessions, using modern technologies like HTML, CSS, JavaScript, PHP, MySQL, and SASS for an integrated, user-friendly solution.

Key Features:

  • Dashboard: Real-time overview of club activities, including the number of tournaments, events, and transactions for the month, plus new subscriber details.
  • Member Management: Interface to add, update, and delete member information, and view new subscriber details like name, email, and phone number.
  • Events: Plan and manage sports events, allowing administrators to create, modify, or cancel events. Users can register and follow upcoming events.
  • Transactions: Track financial transactions with a clear interface to view received payments and expenditures, aiding in precise accounting.
  • Training Sessions: Organize training sessions with an interactive calendar. Users can view schedules and sign up based on availability.
  • Reports: Generate detailed reports on club activities, including participation statistics, financial performance, and other important metrics.
